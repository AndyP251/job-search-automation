# 🚀 Job Scraper — Quick Start Guide

Scrape 22 job boards for SDR/BDR/ADR roles with one click.

---

## Step 1 — Install Python (one time only)

**Already have Python?** Skip to Step 2.

1. Go to **https://www.python.org/downloads/**
2. Click the big yellow **"Download Python"** button
3. Run the installer

> ⚠️ **IMPORTANT (Windows only):**
> Check the box **"Add Python to PATH"** at the bottom of the installer — this is critical!

---

## Step 2 — Setup (one time only)

| Your computer | What to do |
|---|---|
| **Windows** | Double-click **`SETUP.bat`** |
| **Mac / Linux** | Open Terminal, drag `setup.sh` into it, press Enter |

Wait for it to finish (1–2 minutes). You only need to do this once.

---

## Step 3 — Run the Dashboard

| Your computer | What to do |
|---|---|
| **Windows** | Double-click **`RUN.bat`** |
| **Mac / Linux** | Open Terminal, drag `run.sh` into it, press Enter |

Your browser will open to the dashboard at **http://localhost:5050**.

---

## Using the Dashboard

1. **Configure** — adjust role keywords, locations, and the date window (default: 2 days)
2. **Pick sources** — toggle which job boards to scrape (or click "API Only" for a fast run)
3. **Click ▶ Run Scraper** — watch the live progress bar and log
4. **Browse results** — search, filter, and click through to job postings
5. **Download** — grab the `.md` report file

---

## That's it!

- Keep the black terminal window **open** while using the dashboard
- Press **Ctrl+C** in the terminal when you're done
- Run it whenever you want fresh results
